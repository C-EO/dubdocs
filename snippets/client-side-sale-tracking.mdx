You can track the sales on the client-side using `trackSale` method of the [`@dub/analytics` script](/sdks/client-side/introduction).

Here is a quick example of how you can track a sale on the client-side:

<CodeGroup>

```typescript React/Next.js
import { useAnalytics } from "@dub/analytics";

export function CheckoutForm() {
  const { trackSale } = useAnalytics();

  trackSale({
    eventName: "Purchase completed",
    customerExternalId: "CXvG5QOLi8QKBA2jYmDh",
    paymentProcessor: "stripe",
    amount: 5000,
  });
}
```

```javascript Other Frameworks
dubAnalytics.trackSale({
  eventName: "Purchase completed",
  customerExternalId: "CXvG5QOLi8QKBA2jYmDh",
  paymentProcessor: "stripe",
  amount: 5000,
});
```

</CodeGroup>
