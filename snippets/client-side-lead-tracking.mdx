You can track the leads on the client-side using `trackLead` method of the [`@dub/analytics` script](/sdks/client-side/introduction).

Here is a quick example of how you can track a lead on the client-side:

<CodeGroup>

```typescript React/Next.js
import { useAnalytics } from "@dub/analytics";

export function SignUpForm() {
  const { trackLead } = useAnalytics();

  trackLead({
    clickId: "W13FJbgeLIGdlx7s",
    eventName: "Sign Up",
    customerExternalId: "1234567890",
    customerName: "John Doe",
    customerEmail: "john.doe@example.com",
  });
}
```

```javascript Other Frameworks
dubAnalytics.trackLead({
  clickId: "W13FJbgeLIGdlx7s",
  eventName: "Sign Up",
  customerExternalId: "1234567890",
  customerName: "John Doe",
  customerEmail: "john.doe@example.com",
});
```

</CodeGroup>
