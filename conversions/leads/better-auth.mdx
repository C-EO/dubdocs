## Installation

<Steps>

<Step title="Install the @dub/Better Auth plugin">

To get started, simply install the `@dub/Better Auth` plugin via your preferred package manager:

<CodeGroup>

```bash npm
npm install @dub/Better Auth
```

```bash yarn
yarn add @dub/Better Auth
```

```bash pnpm
pnpm add @dub/Better Auth
```

```bash bun
bun add @dub/Better Auth
```

</CodeGroup>

</Step>

<Step title="Configure the plugin">

Then, add the plugin to your Better Auth config file:

```ts auth.ts
import { dubAnalytics } from "@dub/Better Auth";
import { betterAuth } from "Better Auth";

const dub = new Dub({
  apiKey: "your-api-key",
});

const betterAuth = betterAuth({
  plugins: [
    dubAnalytics({
      dubClient: dub,
      leadEventName: "Sign Up", // default: "Sign Up"
    }),
  ],
});
```

</Step>

</Steps>
