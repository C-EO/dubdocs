"og:title": "dub.events.list - Retrieve a list of events - TypeScript SDK"
description: "Retrieve a list of events for the authenticated workspace. The list will be paginated and the provided query parameters allow filtering the returned events."
---

<Note>
  Events endpoints require a [Business plan](https://dub.co/pricing)
  subscription or higher.
</Note>

## Arguments

<ParamField body="event" type="enum" default="clicks">
  The type of event to retrieve events for. Available options: `clicks`,
  `leads`, and `sales`.
</ParamField>

<ParamField body="domain" type="string">
  The domain of the short link.
</ParamField>

<ParamField body="key" type="string">
  The short link slug.
</ParamField>

<ParamField body="linkId" type="string">
  The unique ID of the short link on Dub.
</ParamField>

<ParamField body="externalId" type="string">
  This is the ID of the link in the your database. Must be prefixed with 'ext_'
  when passed as a query parameter.
</ParamField>

<ParamField body="interval" type="enum<string>">
  The interval to retrieve events for.
  
  Available options are `1h`, `24h`, `7d`, `30d`, `90d`, `ytd`, `1y`, and `all`.

Takes precedence over `start` and `end`.

</ParamField>

<ParamField body="start" type="string">
  The start date to retrieve events for.

Cannot be earlier than September 22, 2022.

</ParamField>

<ParamField body="end" type="string">
  The start date to retrieve events for.

Cannot be in the future. If not provided, defaults to the current date.

</ParamField>

<ParamField body="timezone" type="string">
  The IANA time zone code for aligning timeseries granularity (e.g.
  America/New_York). Defaults to UTC.
</ParamField>

<ParamField body="continent" type="string">
  The continent to retrieve events for.

Available options are `AF`, `AN`, `AS`, `EU`, `NA`, `OC`, and `SA`.

</ParamField>

<ParamField body="city" type="string">
  The city to retrieve events for.
</ParamField>

<ParamField body="region" type="string">
  The ISO 3166-2 region code to retrieve analytics for.
</ParamField>

<ParamField body="device" type="string">
  The device to retrieve events for.
</ParamField>

<ParamField body="browser" type="string">
  The browser to retrieve events for.
</ParamField>

<ParamField body="os" type="string">
  The OS to retrieve events for.
</ParamField>

<ParamField body="trigger" type="enum<string>">
  The trigger to retrieve analytics for. Available options: `qr`, `link`. If
  `undefined`, return both QR and link clicks.
</ParamField>

<ParamField body="referer" type="string">
  The referer to retrieve events for.
</ParamField>

<ParamField body="refererUrl" type="string">
  The full referer URL to retrieve events for.
</ParamField>

<ParamField body="url" type="string">
  The URL to retrieve events for.
</ParamField>

<ParamField body="tagId" type="string">
  The tag ID to retrieve events for.
</ParamField>

<ParamField body="tagIds" type="string | string[]">
  The tag IDs to retrieve analytics for. Can be a single string or an array of
  strings.
</ParamField>

<ParamField body="qr" type="boolean">
  Filter for QR code scans. If true, filter for QR code scans only. If false,
  filter for link clicks only. If undefined, return both.
</ParamField>

<ParamField body="root" type="boolean">
  Filter for root domains. If true, filter for domains only. If false, filter
  for links only. If undefined, return both.
</ParamField>

<ParamField body="limit" type="number">
  The number of records to retrieve.
</ParamField>

<ParamField body="offset" type="number">
  Number of records to offset the results by.
</ParamField>

<ParamField body="sortBy" type="enum<'timestamp'>">
  The field to sort the results by.
</ParamField>

<ParamField body="order" type="enum<'asc' | 'desc'>">
  The order to sort the results by.
</ParamField>

<RequestExample>

```ts
await dub.events.list({
  event: "clicks",
  linkId: "abc123",
  start: "a week ago", // we support relative dates like "a week ago"
  end: "today", // we support relative dates like "today"
});
```

</RequestExample>
